## Como Contribuir com Assets

Se você deseja adicionar screenshots ou imagens:

1. Certifique-se de que a imagem não contém informações sensíveis
2. Use formato PNG para screenshots
3. Mantenha resolução adequada (1280x720 ou similar)
4. Nomeie arquivos de forma descritiva (ex: `estrutura-pastas-drive.png`)
